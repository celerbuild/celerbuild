# CelerBuild Pro: For Teams

The Pro version of CelerBuild is designed specifically for team use, enhancing the standard version with advanced features for user management and project organization.

import { Callout } from 'nextra-theme-docs'

<Callout type="info">
    This guide assumes you have subscribed to the Pro plan.
</Callout>

## Key Features

### 1. User Management

**Admin-only feature**

Administrators can:
- Add new users
- Edit user information
- Delete users
- Manage user permissions

### 2. Spaces

**Admin-only feature**

- Create multiple spaces for project organization
- Assign users as space owners
- Facilitate project classification and management

### 3. Project Members

For each project, you can:
- Add members
- Edit member roles

## Getting Started

1. Log in to your CelerBuild Pro account
2. Navigate to the User Management section to set up your team
3. Create Spaces to organize your projects
4. Add members to your projects and assign appropriate roles
