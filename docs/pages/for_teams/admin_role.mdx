# Admin Role in CelerBuild

As an Admin in CelerBuild, you play a crucial role in setting up and managing the system. This guide will walk you through the essential tasks you need to complete after installation.

## Key Responsibilities

1. Adding users
2. Creating spaces
3. Setting up server clusters
4. Adding individual servers

## 1. Adding Users

By default, you'll log in with the admin account. As an administrator, your first task is to create user accounts for your team.

1. Navigate to the Users section
2. Click on "Add User"
3. Fill in the required information:
    - Username
    - Email
    - Password
    - Role (e.g., Developer, Project Owner)
4. Click "Create User" to confirm

![Adding Users Interface](https://github.com/user-attachments/assets/3511225d-a121-4e4b-bcfe-c99c8ab019aa)

## 2. Creating Spaces

Spaces help organize projects and manage access control.

1. Go to the Spaces section
2. Click "Add Space"
3. Enter a name for the new space
4. Assign an owner to the space
5. Click "Create Space"

> **Note**: There's a default space named `default`. You can create additional spaces as needed.

![Creating Spaces Interface](https://github.com/user-attachments/assets/4d44358f-c863-4e6b-8f06-06db8a61975b)

## 3. Setting Up Server Clusters

Server clusters group related servers for easier management.

1. Navigate to the Server Clusters section
2. Click "Add Cluster"
3. Provide a name for the cluster
4. Add any relevant description
5. Click "Create Cluster"

![Adding Server Clusters Interface](https://github.com/user-attachments/assets/c9c933dd-761e-4a3f-9bb4-b663edad3823)

## 4. Adding Servers

After creating clusters, you can add individual servers to them.

1. Go to the Servers section
2. Click "Add Server"
3. Fill in the server details:
    - Server Name
    - Server IP
    - SSH Port
    - Select the appropriate Server Cluster
4. Click "Add Server" to confirm

![Adding Servers Interface](https://github.com/user-attachments/assets/22a3cc9b-b25e-4255-8d06-7090e122e2d8)

## Best Practices

- Regularly review and update user permissions
- Use descriptive names for spaces and server clusters
- Ensure all team members understand their roles and responsibilities
- Keep server information up-to-date and secure

By completing these steps, you'll have set up the basic infrastructure for your team to start using CelerBuild effectively.
