# Prerequisites

Before you start using CelerBuild, you need to complete the following steps:

## 1. Install Dependencies

CelerBuild supports deployment on Linux and macOS environments. Please note that Windows deployment is not supported.

The main dependencies include:

* Installing MariaDB
* Configuring SSH public key

For detailed installation steps and instructions, please refer to the [Dependency Installation Guide](./prerequisites/dependencies).

## 2. Download and Run CelerBuild

After completing the dependency installation, you need to:

* Download the CelerBuild binary file from the official website
* Set execution permissions
* Run CelerBuild

For specific download and running instructions, please check the [Download and Run Guide](./prerequisites/download).

Once you've completed these steps, you can start using CelerBuild. If you encounter any issues during the installation process, please refer to our FAQ or contact us.
